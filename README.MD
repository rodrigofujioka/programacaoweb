1.2 - Baixar e extrair binarios do Maven -> 
         [Maven](https://maven.apache.org/download.cgi)
         
1.3 - Configurar Classpath (Java e Maven) -> 

1.4 - Baixar e instalar ferramentas de desenvolvimento.
